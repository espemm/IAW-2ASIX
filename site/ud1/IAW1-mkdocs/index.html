Què és Mkdocs?¶

Mkdocs és un generador de llocs estàtics que ens permet crear documentació de manera senzilla i ràpida.

Per què utilitzar Mkdocs?¶

Mkdocs és una eina popular per crear documentació de projectes en format de lloc web. Hi ha diverses raons per les quals podries considerar utilitzar Mkdocs:

  • Facilitat d’ús: Mkdocs utilitza un format de fitxer simple i fàcil d’entendre anomenat Markdown per escriure la documentació. Això fa que siga molt accessible per a aquells que no tenen experiència en desenvolupament web.

  • Generació de llocs estàtics: Mkdocs genera llocs web estàtics, cosa que significa que no es requereix un servidor en temps real per mostrar la documentació. Això fa que siga fàcil de distribuir i allotjar a qualsevol servidor web estàndard.

  • Personalització i temes: Mkdocs ofereix una àmplia gamma de temes i opcions de personalització perquè pugues adaptar l’aparença de la documentació segons les teues necessitats i preferències.

  • Suport per a múltiples idiomes: Mkdocs permet crear documentació en diversos idiomes, cosa que és especialment útil si tens una audiència internacional.

  • Integració amb control de versions: Mkdocs es pot integrar fàcilment amb sistemes de control de versions com Git, cosa que facilita la col·laboració i el seguiment de canvis en la documentació.

  • Cerca i navegació intuïtiva: Mkdocs proporciona una funció de cerca incorporada i una navegació intuïtiva, cosa que facilita als usuaris trobar la informació que necessiten ràpidament.

En resum, Mkdocs és una eina poderosa i fàcil de fer servir per crear documentació de projectes. Proporciona una manera senzilla de generar llocs web estàtics amb una aparença personalitzada i ofereix característiques útils com a cerca i navegació intuïtiva. Si cerques una forma eficient de documentar el teu projecte, Mkdocs pot ser una excel·lent opció.

Com instal·leu Mkdocs?¶

Per a instal·lar Mkdocs, necessites tenir Python i pip (el gestor de paquets de Python) instal·lats al sistema. Quan els tingues, pots instal·lar Mkdocs.

En linux amb la següent ordre:

pip install mkdocs

En windows (si no funciona l’ordre anterior):

python -m pip install mkdocs

Instal·lació d’un tema¶

Opcionalment, podeu instal·lar un tema popular com a mkdocs-material per millorar l’aparença de la vostra documentació. Per fer-ho, fes servir la següent ordre:

pip install mkdocs-material
Quan estiga instal·lat, pots iniciar un nou projecte de Mkdocs amb:

mkdocs new nom-projecte
cd nom-projecte
mkdocs serve

Açò iniciarà un servidor local on podràs veure la teua documentació en l’adreça local i port per defecte:

http://127.0.0.1:8000
