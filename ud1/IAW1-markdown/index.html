Què és Markdown?¶

Markdown és un llenguatge de marques lleuger creat per John Gruber i Aaron Swartz que tracta de facilitar la redacció i la lectura de documents en text pla, utilitzant un conjunt d’etiquetes molt senzilles per aplicar estil al document.

Actualment no existeix un estàndard per a Markdown, per aquest motiu existeixen diferents versions o flavors de Markdown.

Treballarem amb el flavor que s’utilitza a GitHub per redactar els fitxers README.md dels repositoris i la documentació tècnica dels projectes.

Markdown Cheat sheet

Sintaxi bàsica Markdown¶

Títol 1 # Títol 1 
Títol 2 ## Títol 2 
Títol 3 ### Títol 3 
Negreta: **bold text** 
Cursiva: *italicized text* 
Cita: > blockquote 
Llista ordenada: 
    1. First item 
    2. Second item 
    3. Third item 
Llista no ordenada: 
    * First item 
    * Second item 
    * Third item 
Codi: 'code' 
Enllaç: [title](https://www.example.com) 
Imatge: ![alt text](image.jpg)
